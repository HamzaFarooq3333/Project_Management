    v.  Develop a procurement plan if external vendors will be involved [Ref 3].

III. **Execution**
    i.  Execute the project plan, managing the work packages [Ref 4].
    ii.  Manage resources effectively [Ref 15].
    iii. Monitor progress against the schedule and budget.
    iv.  Manage project risks and issues.
    v.  Manage changes using a change management process [Ref 3].

IV. **Monitoring and Controlling**
    i.  Regularly track project performance using relevant metrics [Ref 2].
    ii.  Conduct project reviews and audits.
    iii. Manage and control project scope, schedule, budget, and quality [Ref 15].
    iv.  Manage stakeholder communication and expectations.
    v.  Address any changes through an established change control process [Ref 3].
