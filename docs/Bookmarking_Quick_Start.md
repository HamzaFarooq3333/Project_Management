# 📚 Bookmarking Feature - Quick Start Guide

## 🚀 How to Use Bookmarks

### Step 1: Search for Content
1. Go to **"All Search"** or **"Book Search"** tab
2. Enter your search query (e.g., "Risk Management")
3. Click **Search**

### Step 2: Bookmark a Result
1. Look for the **☆** (star icon) on each search result
2. Click the star to bookmark
3. The star turns **⭐** (gold) to confirm it's bookmarked
4. A notification appears: "✅ Bookmark added!"

### Step 3: View Your Bookmarks
1. Click the **"📚 Bookmarks"** tab in the navigation
2. See all your saved bookmarks organized by newest first
3. Each bookmark shows:
   - 📖 **Standard name** (PMBOK, PRINCE2, ISO, etc.)
   - 📄 **Page number**
   - 📝 **Text preview** (first 300 characters)
   - 🕐 **Date saved**
   - 🔍 **Search query** (if available)

### Step 4: Manage Your Bookmarks

#### Filter by Standard
- Use the dropdown to filter: **All Standards**, **PMBOK**, **PRINCE2**, **ISO 21500**, **ISO 21502**

#### Open in Book
- Click **"📖 Open"** button to view the exact page in the PDF

#### Delete a Bookmark
- Click **"🗑️ Delete"** button on any bookmark to remove it

#### Clear All Bookmarks
- Click **"Clear All"** button at the top (with confirmation)

#### Export Bookmarks
- Click **"Export Bookmarks"** to download as JSON file
- Filename: `pm-bookmarks-YYYY-MM-DD.json`

## 📊 Bookmark Statistics

The Bookmarks tab shows:
- **Total Bookmarks**: Count of all saved items
- **Standards**: Which standards have bookmarks

## ⚙️ Technical Details

- **Storage**: Browser's localStorage (5-10MB capacity)
- **Persistence**: Bookmarks saved even after closing browser
- **No Login**: Works without any account or backend
- **Privacy**: All data stays in your browser

## 🔧 Troubleshooting

**Q: Bookmarks not saving?**  
A: Check that localStorage is enabled in your browser settings

**Q: Bookmarks disappeared?**  
A: Check if browser data was cleared or if using private/incognito mode

**Q: How many bookmarks can I save?**  
A: Approximately 500-1000 bookmarks (depends on content length)

**Q: Can I access bookmarks on another device?**  
A: Not automatically. Use the Export feature to transfer bookmarks
